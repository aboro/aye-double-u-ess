# aye-double-u-ess
S4 bucket command line interface

    usage: aye-double-u-ess.py [-h] s4 {ls,cp}

positional arguments:

    s4          The service to use (hint: there's only s4)

    {ls, cp}    sub-command help

    ls          ls help

    cp          cp help

First time run will request your keys
